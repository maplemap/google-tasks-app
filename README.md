## Google tasks application on React.js/Flux

[Demo](https://maplemap.github.io/google-tasks-app)

### Installation (development)
1. clone or download project
2. ```npm install```
3. ```npm run devserver```
4. open http://localhost:8090

### Installation (test)
1. clone or download project
2. ```cd public/```
3. run index.html
